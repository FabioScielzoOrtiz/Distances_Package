1) To create a folder PACKAGE with the following structure:

PACKAGE/
    setup.py
    README.md
    LICENSE
    example_package_FabioScielzoOrtiz/
        __init__.py
        example.py

2) To put in each file the following:

In example.py :

def add_one(number):
    return number + 1


In __init__.py :

from .example import add_one

In setup.py :

from setuptools import setup, find_packages

with open("README.md", "r", encoding="utf-8") as fh:
    long_description = fh.read()

setup(
    name="example_package_FabioScielzoOrtiz",
    version="0.0.1",
    author="Fabio Scielzo Ortiz",
    author_email="fabioscielzo98@gmail.com",
    description="A small example package",
    long_description=long_description,
    long_description_content_type="text/markdown",
    url="https://github.com/FabioScielzoOrtiz/Distances_Package",  # The link of the girhub repository there your code is.
    packages=find_packages(),
    classifiers=[
        "Programming Language :: Python :: 3",
        "License :: OSI Approved :: MIT License",
        "Operating System :: OS Independent",
    ],
    python_requires=">=3.7",
)


3) You have to upload the folder PACKAGE to a GitHub repository, and paste its link in setup.py as was shown before (url="https://github.com/FabioScielzoOrtiz/Distances_Package").

4) To deploy your package in testpypi run in the terminal the following lines of code:

python -m pip install --upgrade setuptools wheel

python setup.py sdist bdist_wheel

python -m pip install --upgrade twine
